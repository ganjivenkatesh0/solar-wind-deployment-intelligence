# Dataset Summary

This document summarizes the expected datasets for the Solar & Wind Deployment Intelligence Platform.

## Expected Datasets

### NASA POWER
- Purpose: Solar resource and weather prediction.
- Expected data: solar irradiance, temperature, humidity, wind speed.
- Status: Pending download.

### Global Wind Atlas
- Purpose: Wind resource assessment.
- Expected data: wind speed, wind direction, wind density.
- Status: Pending download.

### Sentinel-2
- Purpose: Satellite imagery for land cover analysis.
- Expected data: vegetation, water bodies, land cover features.
- Status: Pending download.

### OpenStreetMap (OSM)
- Purpose: Infrastructure mapping.
- Expected data: roads, buildings, substations, connectivity.
- Status: Pending download.

### SRTM
- Purpose: Terrain and elevation analysis.
- Expected data: elevation, slope, terrain features.
- Status: Pending download.

## Notes

- The dataset summary will be updated after the actual files are downloaded and analyzed.
- Future updates will include data schema, file format, and preprocessing notes.
